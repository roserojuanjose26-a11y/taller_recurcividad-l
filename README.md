 Instrucciones de Ejecución (NetBeans):
 
Para visualizar los resultados de cada ejercicio, hay que seguir estos pasos:

Clonar o descargar este repositorio.
Abrir el proyecto Recursividad desde NetBeans.
Expandir la carpeta Source Packages y localizar el paquete deseado.
Ejecución individual:
Haga clic derecho sobre el archivo .java que desea probar.
Seleccione la opción Run File.
Revisar la pestaña Output en la parte inferior de NetBeans para interactuar con el programa.
